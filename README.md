# 한국다봄

한국 여행 플래너 백엔드 프로젝트 입니다.
OPEN API를 이용하여 여행지 데이터를 가져와 보여주고 여행 일정을 생성/관리합니다.

## 1. 사용 기술

-   Java 8 -> 17
-   Spring Boot
-   Spring JDBC -> MyBatis
-   Spring Security
-   MySQL 5.7
-   Lombok
-   Junit, Mockito

## 2. 주요 기능

-   사용자 인증/인가(JWT)
-   OPEN API를 이용한 여행지 탐색
-   여행 일정 생성
-   여행지 리뷰
-   최단 경로 탐색 기능(제주도)
